## Линейная регрессия

Линейная регрессия - это статистический метод, используемый для моделирования взаимосвязи между зависимой переменной и одной или несколькими независимыми переменными. **Метод поиска наилучшей прямой линии через множество точек на графике. Эта линия становится инструментом для прогнозирования новых значений.**

Линейная модель связывает переменные входа и выхода с помощью линейных операций - умножение и сложение. 
Работать эта модель может в пространстве любой размерности. Векторные входы и векторные выходы. Линейная регрессия описывается формулой:
![[Pasted image 20260218184623.png]]
x - вектор входных признаков, y - вектор выходных признаков, A - матрица весовых коэффициентов, б - дополнительный вектор обучаемых параметров (вектор смещения).

"Линейная" = линейная по параметрам:

Нужно подобрать матрицу и вектор смещения. Нужно придумать функцию ошибки и ее минимизировать. Для задач регрессии, где используется линейная регрессия, а ее функция ошибки - среднеквадратичная ошибка (MSE).
![[Pasted image 20260222144854.png]]
Такую функцию ошибки легко минимизировать.
1) Неитерационный МНК (**нелинейный метод наименьших квадратов**)
   метод нахождения коэффициентов линейной регрессии, который не требует постепенного подбора (итераций), а вычисляет коэффициенты мгновенно по формуле.
   ![[Pasted image 20260222161343.png]]
   На выходе этой формулы получаем вектор, который содержит значение смещения и затем веса при входных переменных.
   ![[Pasted image 20260222161907.png]]
   Матрица X представляет собой:![[Pasted image 20260222162211.png]]. К признакам слева добавляется столбец единиц, чтобы искомая прямая не всегда проходила через точку 0, а имела смещение.
   Библиотека sklearn использует этот метод наименьших квадратов для вычисления.
   **Используется только для линейной регрессии**
2) Итерационный МНК. Градиентный спуск. Можно использовать и для других моделей.
   Изменять параметры в направлении противоположном от градиента. 
   Этот метод не используется для простых линейных моделей, так как затрачивает больше времени чем нелинейный МНК.
