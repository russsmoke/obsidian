# Description
Центрирование данных используется широко в машинном обучении, так как упрощает обработку данных. Каким образом: при центрировании данных, из исходных признаков вычитаются их среднее значение, чтобы полученные результаты имели нулевое среднее. Центрирование не меняет структуру данных (расстояния остаются теми же), оно лишь "центрирует" вокруг нуля. 
Например, есть данные X = \[10,20,30\]. Их среднее равно 20, после центрирования получаем X = \[-10,0,10\]. Как видно из примера расстояние не поменялось после центрирования, зато мы теперь явно видим отношения между данными.
