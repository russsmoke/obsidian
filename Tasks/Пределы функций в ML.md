## Методы вычисления пределов
1) Элементарные функции в точке из D(f).
   Если f(x) - элементарная, а точка x0 принадлежит D(f), тогда предел в точке равна f(x0)
2) Пристальный взгляд.
   "Прикинуть", к какому значению стремится функция по указанной базе и сразу найти ответ, если не возникает сложных неопределенностей.
3) Деление на старший (неопределенность с бесконечностью)/младший (неопределенность с 0) член. Для многочленов и для показательных функций.
4) Домножение на сопряженное. ![[Pasted image 20260124165129.png]]
5) [[Символика Ландау]] и связанные обозначения
   ![[Pasted image 20260124172342.png]]
   Функция f(x) называется финально ограниченной по базе x0, если в некоторой окрестности точки x0 она ограничена. (f(x) = 1/x по базе x0=5, функция ограничена в окрестности этой точки).
   Функция f(x) есть O(g(x)) по базе $x \to x_0$, если f(x)/g(x) является финально ограниченной по этой же базе.
