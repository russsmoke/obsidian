# Description
Задача, которая была решена необычным способом.
Названа в честь ведущего американского шоу "Давайте заключим сделку". Эта задача затрагивалась в этом самом шоу. Рассмотрим задачу с дополнительными условиями.

Участнику предлагается выбрать одну из трех дверей. За одной из них находится автомобиль, за двумя другими - козы. Когда участник выбрал одну из дверей, ведущий, зная за какой дверью автомобиль, открывает другую дверь, где находится коза. Затем спрашивает, не хочет ли участник изменить свой выбор, поменять выбранную дверь. И тут возникает парадокс, нужно ли менять свой выбор, чтобы повысить шансы на выбор правильной двери.
Интуитивно может показаться, что никакой разницы нет, потому что шансы 50 на 50. Однако, если посчитать математически, применяя теорему Байеса, то становится понятно, что поменяв выбор двери, шансы становятся выше (с 1/3 до 2/3).
