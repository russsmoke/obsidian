# Description
## Создание графика Matplotlib
Основные компоненты:
1) Данные: могут быть представлены в виде списков, массивов или данных из внешних источников. Тип данных влияет на график и способ его настройки.
2) Тип графика: линейные графики, столбчатые диаграммы, диаграммы рассеивания, гистограммы.
3) Настройки: можно изменять цвета, маркеры, стили линий, метки, заголовки, свойства осей.
4) Отображение и сохранение: можно отобразить на экране или сохранить файл в различных форматах (например, png, pdf)

## Фигура
Базовый компонент графика. Представляет пространство или контейнер, где будут отображаться графики и диаграммы. Отслеживает все дочерние оси графика, все его компоненты - заголовки, легенды, подписи, вложенные подфигуры.

Для создания фигуры:
`plt.figure()`
Отрисовка фигуры:
`plt.show()`

При первом вызове функции для построения графика библиотека проверяет наличие текущей активной фигуры, если ее нет она создается автоматически.

### Настройка фигуры
`plt.figure(num=None, figsize=None, dpi=None, *, facecolor=None, edgecolor=None, frameon=True, FigureClass=matplotlib.figure.Figure, clear=False, **kwargs)`

- `num` - уникальный идентификатор фигуры в виде целого числа или строки. Одновременно может существовать несколько фигур. И если фигура с таким идентификатором уже существует, она становится активной и возвращается. Если фигуры с таким идентификатором нет, создается новая, становится активной и возвращается.
- `figsize` - задает размер фигуры в виде кортежа `(float,float)`, который представляет значения ширины и высоты в дюймах (по умолчанию `[6.4,4.8]`)
- `dpi` - разрешение фигуры в точках на дюйм. По умолчанию 100.0
- `facecolor`- цвет фона
- `edgecolor - цвет границы
- `frameon` - рамка фигуры. Если false, отключается отрисовка рамки фигуры. По умолчанию true
- `FigureClass` - тип фигуры.
- `clear` - если True и фигура существует, то она очищается. По умолчанию False
- `layout` - компоновка
	- `constrained` - размеры осей корректируются, чтобы избежать перекрытия элементов оформления осей. Сложные макеты графиков и цветовые шкалы.
	- `compressed` - тот же алгоритм что в первом, но удаляет лишнее пространство между осями с фиксированным соотношением сторон. Для простых сеток осей.
	- `tight` - корректирует параметры подграфиков таким образом, чтобы элементы оформления не перекрывались.
	- `none` - не применяется

## Оси
На каждой фигуре есть одна или несколько осей. Ось представляет собой отдельный график или подграфик внутри фигуры в виде объекта `Axes`. Представляют пространство, где можно визуализировать данные, устанавливать системы координат и применять различные настройки с помощью методов типа Axes.

Для создания одного набора осей на графике можно использовать функцию `subplots()`

### Создание графика
Для создания графика у объекта `Axes` применяется функция `plot()`:
`plot(*args, scalex=True, scalex=True, data=None, **kwargs)`
- `args` - передаются значения для осей графика. Если передаются два набора данных, то первый набор для оси X, второй набор для оси Y. Можно также передавать только один набор данных для оси Y.
- `scalex` и `scaley` - определяют, будет ли график масштабироваться. 
- `data` - представляет объект с данными для имен меток для построения графика по осям x и y

## Два типа интерфейсов
1) Построение графика с помощью методов объекта `Axes`:
   ```
   fig, axes = plt.subplots()
   axes.plot(x, y)
   axes.set_title("График")
   plt.show()
   ```
   2) Функциональный стиль:
```
plt.plot(x, y)
plt.title("График")
plt.show()
```
## Функция plot()
Принимает два аргумента - например, списки. Сначала для отображения по оси Х и затем по оси Y и с помощью функции **show()** отображаем график.

## Метки осей
Для осей можно указать метки.
1) `Axes`
   - `set_xlabel(xlabel (текст метки), fontdict=None, labelpad=None (расстояние до оси по умолчанию 4.0) , *, loc=None, **kwargs)`
   - `set_xlabel(xlabel (текст метки), fontdict=None, labelpad=None (расстояние до оси по умолчанию 4.0) , *, loc=None, **kwargs)`

   - `loc (устанавливает положение метки. Для метки по оси Х применяются значения left, center, right. Для метки по оси Y применяются значения bottom, center, top. В обоих случаях значение по умолчанию center)`
2) Функциональный вид
   - `matplotlib.pyplot.xlabel(xlabel, fontdict=None, labelpad=None, *, loc=None, **kwargs)`
   - `matplotlib.pyplot.ylabel(ylabel, fontdict=None, labelpad=None, *, loc=None, **kwargs)`
### Настройка текста
- `color` - цвет текста в виде названия или шестнадцатеричного значения
- `fontfamily` - семейство шрифта. Возможные значения: "serif", "sans-serif", "cursive", "fantasy", "monospace" или название конкретного шрифта
- `fontsize` \ `size` - размер шрифта в виде числового значения с плавающей точкой или "xx-small", "x-small", "small", "medium", "large", "x-large", "xx-large"
- `fontstretch` \ `stretch` - растяжение шрифта в виде значения в диапазоне 0-1000 или в виде одной из следующих констант `"ultra-condensed", "extra-condensed", "condensed", "semi-condensed", "normal", "semi-expanded", "expanded", "extra-expanded", "ultra-expanded"`
- `fontstyle` \ `style` - стиль шрифта в виде одного из следующих значений "normal", "italic", "oblique"
- `fontweight` \ `weight` - толщина шрифта в виде значения в диапазоне 0-1000 или одного из значений "ultralight", "light", "normal", "regular", "book", "medium", "roman", "semibold", "demibold", "demi", "bold", "heavy", "extra bold", "black"
