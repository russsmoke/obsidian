# Description
Статистический метод, который используется для сглаживания данных, чтобы лучше видеть **тренды**, убирая краткосрочные колебания и шум.
## Как вычисляется
Есть ряд данных: x1, x2, x3, x4, x5, ...
Выбирается размер **скользящего окна** k (например, 3)
**Скользящее среднее для момента t** - это среднее арифметическое последних k значений:
![[Pasted image 20260121212203.png]]
Среднее **скользит** по данным, сглаживая резкие скачки.
Но лучше использовать [[Экспоненциальное скользящее среднее (EMA)]].
**Скользящее среднее — это "умное усреднение"**, которое постоянно обновляется, отслеживая недавние значения и постепенно забывая старые.

**В оптимизаторах (RMSProp, Adam):**  
Оно нужно, чтобы **адаптировать размер шага** для каждого параметра, глядя на историю его градиентов:

- Если градиент "прыгал" → уменьшаем шаг.
- Если градиент был стабильным → увеличиваем шаг.

Это делает обучение нейронных сетей **быстрее и стабильнее**.
