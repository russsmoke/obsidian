# Description
Она охватывает три отдельные, но пересекающиеся сферы:
1. Навыки специалиста по математической статистике, который умеет моделировать наборы данных и извлекать из них основное.
2. Навыки специалиста компьютерных наук, умеющего проектировать алгоритмы для эффективного хранения, обработки и визуализации этих данных.
3. Экспертные знания предметной области, полученные в ходе традиционного изучения предмета, уметь как задавать правильные вопросы, так и формулировать ответы в соответствующем контексте.
==То есть, наука о данных это не новые знания, а новый набор навыков.==
