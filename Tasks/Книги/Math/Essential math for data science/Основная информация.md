# Описание
Такие темы как линейная алгебра, статистика, теория вероятности нужны для профессии аналитика данных, а точнее для машинного обучения. Именно математика лежит в основе всех моделей для обработки, сбора данных. 

С этой книгой читатель научится применять знания по математике, статистике и прочему на реальных задачах. 

Нужные библиотеки python: numpy, sympy, scipy, sklearn.
