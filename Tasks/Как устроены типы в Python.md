# Description
Язык питон обладает динамической типизацией, то есть тип переменных определяется динамически, а не явно. 
Стандартная реализация питона написана на С. Каждый объект питон это замаскированная структура языка С. Целочисленная переменная на языке С - это метка, определяющая место в памяти (байты). А в питоне целочисленная переменная - это указатель на место в памяти, где хранится вся информация об объекте языка питон, включая байты, тип и счетчик ссылок. (когда создается новый объект с таким же именем, эта переменная ссылается теперь на этот объект, объекты, которые не используются, удаляются). 

В питон можно даже создавать неоднородный списки с разными типами переменных. Реализация выглядит так: указатель на список содержит указатель на блок указателей на отдельный объекты питон (каждый хранит свою избыточную информацию). Также есть массивы с фиксированным числом элементов. Например, array (нужно импортировать). Гораздо удобнее использовать массив фиксированной длины из библиотеки [[Библиотека NumPy|нампай]] (ndarray), так как предоставляет возможность использовать удобные операции над массивами.
