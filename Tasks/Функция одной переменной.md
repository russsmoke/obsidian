Функция - отображение (соответствие) элементов одного множества в элементы другого множества.
Функция - правило, по которому связаны два множества

**Образ**
Образ множества А для функции f - это множество **значений**, которые принимает f при условии x e A
**Прообраз**
Прообраз множества B для функции f - это множество элементов области **определения**, при которых f принимает значения из B

**Композиция функций** - возникает в тот момент, когда мы вместо аргумента (переменной) одной функции подставляем другую функцию

**Элементарные функции:**
Такие, которые можно получить с помощью конечного числа арифметических действий и композиций из следующих основных элементарных функций.
1) степенная функция с любым действительным показателем
2) показательная и логарифмическая функция, тригонометрическая
