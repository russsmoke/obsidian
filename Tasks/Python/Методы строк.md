[[Python]]
1) `str.capitalize()` - возвращает копию строки, у которой первая буква заглавная, а остальные приведены к строчным
2) `str.count(sub)` - возвращает количество неперекрывающихся вхождений подстроки sub.  К примеру, если искать в строке «ААААА» неперекрывающиеся значения «АА», то первое вхождение будет «AAAAA». Второе — «AAAAA». Больше неперекрывающихся вхождений нет. Так, поиск последующих вхождений подстроки происходит с индекса, который следует за последним найденным вхождением
3) `str.endswith(suffix)` - возвращает True, если строка оканчивается на подстроку suffix. `suffix` может быть кортежем проверяемых окончаний строки
4) `str.find(sub)` - возвращает индекс первого вхождения подстроки sub. Если подстрока не найдена возвращает -1.
5) `str.index(sub)` - возвращает индекс первого вхождения подстроки sub. Вызывает исключение ValueError, если подстрока не найдена.
6) `str.isalnum()` - возвращает True, если все символы строки являются **буквами и цифрами** и в строке есть хотя бы один символ (Пробел и остальные символы буду вызвать false).
7) `str.isalpha()` - возвращает True, если все символы строки являются буквами и в строке есть хотя бы один символ.
8) `str.isdigit()` - возвращает True, если все символы строки являются цифрами и в строке  есть хотя бы один символ.
9) `str.islower()` - возвращает True, если все буквенные символы строки написаны строчными буквами.
10) `str.isupper()` - возвращает True, если все буквы в строке большие и есть хотя бы один символ
11) `str.join(str_col)` - Возвращает строку, полученную конкатенацией (сложением) строк — элементов коллекции `str_col` (обозначение коллекции с элементами типа данных «строка»). Разделителем является строка, для которой вызван метод.
    Пример:
    ```
    a = ["1","2","3"]
    "; ".join(a)       #"1; 2; 3"
    ```
12) `str.ljust(width, fillchar)` - возвращает строку длинной width с выравниванием по левому краю. Строка дополняется справа символами fillchar до требуемой длины. По умолчанию, fillchar - строка.
    ```
    s = "text"
    s.ljust(10,"=")       # "text======"
    ```
13) `str.lower()` - возвращает копию строки, у которой все буквы приведены к нижнему регистру.
14) `str.lstrip(chars)` - возвращает строку, у которой в начале удалены символы, встречающиеся в строке chars. Если значение chars не задано, то пробельные символы удаляются.
    ```
    s = "BCCAstring"
    s.lstrip("ABC")       # "string"
    ```
