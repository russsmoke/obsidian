[[Python]]
Для того, чтобы сократить количество строк кода, можно использовать в циклах моржовый оператор `:=`
Вместо:
```
name = input("Введите имя: ")
while name != "СТОП":
    print(f"Здравствуйте, {name}")
    name = input("Введите имя: ")
print("Завершение программы.")
```
Можно сократить код:
```
while (name := input("Введите имя: ")) != "СТОП":
	print(f"Здравствуйте, {name}")
print("Завершение программы.")
```
